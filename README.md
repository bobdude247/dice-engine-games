# dice-engine-games

Reusable dice engine plus dice-based games (starting with Five of a Kind and Craps).

## Structure
- src/engine/dice: pure roll/scoring primitives
- src/games/five-of-a-kind: reusable scorecard-style game
- src/games/craps: table rules/state machine
- src/ui: optional UI adapters
- tests: engine/game tests
